Get rid of the secondary adminbar menu (from the right) and show it only on click. No more accidental clicks on the profile!

== Description ==

A plugin which adds a toggle button for the secondary WordPress admin bar which can be easily clicked by accident in WP5.0

== Installation ==

To install the plugin get to your WordPress Dashboard, then navigate to the Plugins->Add New menu. Once the pages loads, search for WP Adminbar Secondary Relocator and click the Install and then Activate button. Once the plugin is installed and activated, you will see the top-right menu bar disappear and an arrow icon instead of it. You can show it if you want by clicking the arrow button.
